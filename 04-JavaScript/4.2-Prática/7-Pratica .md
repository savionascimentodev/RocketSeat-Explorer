- Math:
  - **Pacote** com funcionalidades matemáticas ✅;

  - Math.ceil() **Arredonda para cima**✅;
  - Math.floor() **Arredonda para baixo** ✅;
- Tipo de dado: **Nan** ✅;
- Tipo de erro: **SyntaxError** (erro de sintaxe) ✅;
- Console:
